# CodeGig

## Description

Simple job finder app for coders. This app uses Node, Express, Sequelize (Postgres) and Handlebars. 

## Installation

``` bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start

# Dev Server (Nodemon)
npm run dev
```
